== Description ==

A simple plugin allows you manage your client's testimonial.

You can insert your client's testimonial and display it at front-end by slider or grid.

Shortcode [c4d-testimonial-slider count="10"][/c4d-testimonial-slider]

This plugin is used in C4D Vision - Responsive  WooCommerce Theme

You can view here http://coffee4dev.com/responsive-woocommerce-theme-c4d-vision/

== Installation ==

1. Upload `c4d-testimonial` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
